# BINF-5507-Summer2026
Course page for the Clinical Bioinformatics ML/AI Course for Summer 2026. Every week, this page will get updated with new material and fun coding stuff.

## 📚 Labs

### [Lab 1: Getting Set Up with GitHub!](Labs/Lab%201) 🎉
Learn the basics of GitHub and VS Code integration. Create your first repository, clone it to your local machine, and make your first commit. Perfect for absolute beginners!

### [Lab 2: The Joys of Data Curation and Preprocessing!](Labs/Lab%202/) 
Discover the joy of turning a messy dataset into something that can used for modeling purposes. Learn different techniques for data preprocessing.

### [Lab 3: Regression: Linear Relationships between Features and Outcomes.](Labs/Lab%203/) 
Fit linear and logistic models and use the models to infer on new data.

### [Lab 4: Classification: k-Nearest Neighbours, Decision Trees, Random Forests and Support Vector Classifiers, oh my!](Labs/Lab%204/)
Fit different classification models to the data; compare and contrast which offer better predictive performance.
